## Architecture Decisions

- Built with the Next.js App Router for optimized routing and performance.
- Separated UI into reusable components for scalability.
- Implemented state-driven rendering using React hooks.
- Simulated asynchronous AI processing using controlled timeout logic.
- Managed audio playback using `useRef` to prevent overlapping sounds.
- Designed UI with Tailwind utility classes for consistent styling.
